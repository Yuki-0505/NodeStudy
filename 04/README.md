# 1. `express`

## 1.1. 安装

```
npm install express --save
```

# 2. `nodemon`修改完文件自动重启

## 2.1. 安装

```
npm install --global nodemon
```

## 2.2. 检测版本

```
nodemon --version
```

# 2. `Express`中使用`art-template`

## 2.1. 安装

```
npm install --save express-art-template
```

# 3. 使用`body-parser`获取 POST 请求数据

## 3.1. 安装

```
npm install --save body-parser
```
